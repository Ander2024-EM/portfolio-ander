## Sobre el proyecto

Este portafolio fue diseñado como una experiencia moderna y profesional enfocada en:
- presentación visual premium;
- motion design elegante;
- responsive avanzado;
- storytelling visual;
- experiencia de usuario moderna;
- arquitectura frontend limpia;
- rendimiento y accesibilidad.

- Navegación moderna y responsive.
- Motion system con Framer Motion.
- Modal avanzado para proyectos.
- Experiencia premium en móvil y desktop.
- Microinteracciones y transiciones suaves.
- Diseño minimalista oscuro.
- Proyectos organizados tipo case study.
- Storytelling visual para presentación de proyectos.
- Optimización SEO básica.
