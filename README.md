# Portafolio Ander Eli

Portafolio profesional construido con Next.js para presentar proyectos reales de desarrollo backend, Android, Java Swing, bases de datos y sistemas administrativos.

## Tecnologías

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- Lucide React

## Contenido principal

- Hero profesional con enlaces de contacto.
- Proyectos destacados con capturas reales.
- Galería modal por proyecto.
- Filtros por tipo de proyecto.
- Stack tecnológico, servicios, trayectoria y contacto.
- Metadata SEO en español para mejorar la presentación al compartir el sitio.

## Ejecutar localmente

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) para ver el portafolio.

## Validación

```bash
npm run lint
npm run build
```
